# 🧠 Neural Network from Scratch (NumPy)

## 📌 Overview
This project implements a simple neural network from scratch using NumPy, without any machine learning libraries like TensorFlow or PyTorch.

## 🎯 Objective
Solve the XOR problem using a multi-layer neural network.

## 🏗 Architecture
- Input Layer: 2 neurons
- Hidden Layer: 4 neurons
- Output Layer: 1 neuron

## ⚙️ Concepts Used
- Forward Propagation
- Backpropagation
- Sigmoid Activation Function
- Gradient Descent
